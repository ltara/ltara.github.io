MongoDB 学习之基本操作（Node）

下载与安装

  1. 下载地址
  2. 安装：全部默认下一步就行了
  3. 安装成功
# 在命令行输入以下命令
mongod --version

成功：

配置

环境变量path的配置：

  1. 复制bin目录

MongoDB默认会安装在C盘下的Program Files目录下

  1. 新建环境变量

启动与关闭

  • 启动
    在命令行里输入mongod即可开启

  • 关闭
    在命令行里按Ctrl + C即可

连接和断开数据库

连接数据库

  1. 在C盘目录下创建一个目录/data/db作为存储目录
  2. 在新的命令行窗口（数据库启动窗口不关）输入mongo即可
  3. 连接成功

断开数据库

在命令行窗口输入exit即可

概念

  • 数据库
  • 集合（在MySQL中叫数据表）
  • 文档（在MySQL中叫记录）

存储结构：

基本命令

  • show dbs
    查看数据库目录
  • db
    当前所在数据库
  • show collections
    查看当前数据库集合目录
  • use 任意数据库名称（可以不存在）
    切换到指定数据库，如果不存在会新建一个数据库，但是不会立即新建，只有当该数据库中有数据时，即添加文档，才会建立
  • db.collection.insert()
    添加数据（文档），collection代指某一集合，要使用具体的集合名称，以下collection同义
  • db.collection.find()
    查找数据（文档）
  • db.dropDatabase()
    删除当前所在数据库
  • db.collection.drop()
    删除集合

MongoDB 在 Node 中的使用

使用 Mongoose 第三方库来使用 MongoDB

示例

const mongoose = require('mongoose');
mongoose.connect('mongodb://localhost:27017/test', {useNewUrlParser: true, useUnifiedTopology: true});

const Cat = mongoose.model('Cat', { name: String });

const kitty = new Cat({ name: 'Zildjian' });
kitty.save().then(() => console.log('meow'));

快速使用

配置
const mongoose = require('mongoose');
const Schema = mongoose.Schema;

//连接本机数据库
mongoose.connect('mongodb://localhost:27017/test', { useNewUrlParser: true, useUnifiedTopology: true });

//设计集合结构
const userSchema = new Schema({
    username: {
        type: String,
        required: true
    },
    password: {
        type: String,
        required: true
    },
    email: {
        type: String
    }
});

//将文档结构发布为模型
const User = mongoose.model('User', userSchema);
增（添加数据）
//实例化一个User对象admin
const admin = new User({
    username: 'admin',
    password: '123456',
    email: 'admin2@admin.com'
});
const admin2 = new User({
    username: 'admin2',
    password: '1234567',
    email: 'admin2@admin.com'
});

// 将admin对象存储到数据库
admin.save((err, ret) => {
    if (err) {
        return console.log(err);
    }
    console.log(ret, '成功');
});
查（查询数据）
// 查询所有
User.find((err, ret) => {
    if (err) {
        return console.log(err);
    }
    console.log(ret); //查询结果
});

// 按条件查询所有
User.find({
    username: 'admin2'
}, (err, ret) => {
    if (err) {
        return console.log(err);
    }
    console.log(ret); //查询结果
});

// 按条件查询单个
User.findOne({
    username: 'admin2'
}, (err, ret) => {
    if (err) {
        return console.log(err);
    }
    console.log(ret); //查询结果
});
删（删除数据）
// 删除单个数据
User.deleteOne({ 
    username: 'admin2'    
}, function (err) {
    if (err) {
        return console.log(err);
    }
    console.log('删除成功');
});

// 删除所有符合条件的数据
User.deleteMany({
    username: 'admin2'
}, (err) => {
    if (err) {
        return console.log(err);
    }
    console.log('删除成功');
});

//旧版本API
User.remove({
    username: 'admin2'
}, (err) => {
    if (err) {
        return console.log(err);
    }
    console.log('删除成功');
});
改（更新数据）
User.updateOne({
    username: 'admin'
},{
    password: '12345'
}, (err, ret) => {
    if (err) {
        return console.log(err);
    }
    console.log(ret); //返回操作结果（成功与否）
});

User.findOneAndUpdate({
    username: 'admin'
},{
    password: '123'
}, (err, ret) => {
    if (err) {
        return console.log(err);
    }
    console.log(ret); //返回更新之前的数据
});
